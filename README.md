# 📋 Survey App

A full-featured survey web application built with vanilla HTML, CSS, and JavaScript.

Built by **Leon Njoroge** ([@lnwleon](https://github.com/lnwleon)) · Nairobi, Kenya

---

## 🚀 Features

- **Two roles**: Survey Coordinator (admin) and Survey Respondent
- Coordinators can create surveys with up to 10 questions and 5 choices each
- Coordinators can open and close surveys
- Respondents can take any open survey
- Results shown as a data table + visual bar charts
- All data persisted in `localStorage` — no backend needed
- Dark tech UI built from scratch

## 🛠 Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- No frameworks, no build step
- `localStorage` for data persistence

## 📸 How to Use

1. Open `index.html` in any browser
2. As a **respondent**: click any open survey and fill it in
3. As a **coordinator**: click **Coordinator Login** (demo: `admin` / `admin123`)
   - Create surveys, open/close them, view results
